# Telegram-бот @Assist_SupportBot

Бот для продажи доступа к каналу «Скорая помощь для ассистента»

## Функционал

- ✅ Приветствие с именем пользователя
- ✅ Информация о тарифах (БАЗОВЫЙ и АССИСТЕНТ ДЛЯ АССИСТЕНТА)
- ✅ Информация об авторах канала
- ✅ Контакт менеджера для оформления (@sp_assistant)
- ✅ База данных для хранения статистики пользователей
- ✅ Кнопки навигации "В меню" и "Назад"

## Структура проекта

```
tg_bot_Assist/
├── bot.py                  # Главный файл запуска
├── config.py              # Конфигурация
├── requirements.txt       # Зависимости Python
├── .env                   # Переменные окружения (создать вручную)
├── .env.example           # Пример файла .env
├── handlers/              # Обработчики команд и кнопок
│   ├── start.py           # Команда /start
│   └── callbacks.py       # Обработка нажатий кнопок
├── keyboards/             # Клавиатуры (кнопки)
│   └── inline.py          # Inline-кнопки
├── texts/                 # Тексты сообщений
│   └── messages.py        # Все тексты из ТЗ
└── database/              # База данных
    └── db.py              # Работа с SQLite
```

## Установка и запуск

### 1. Требования

- Python 3.8 или выше
- Telegram-аккаунт
- Токен бота от @BotFather

### 2. Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `Assist Support Bot`)
4. Введите username бота (должен заканчиваться на `bot`, например: `Assist_SupportBot`)
5. Скопируйте токен, который выдаст BotFather (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 3. Установка зависимостей

Откройте терминал (командную строку) и перейдите в папку проекта:

```bash
cd "d:\Danila AI\projects\active\tg_bot_Assist"
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

### 4. Настройка

Создайте файл `.env` на основе `.env.example`:

```bash
copy .env.example .env
```

Откройте файл `.env` и вставьте ваш токен:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

### 5. Запуск бота

Запустите бота командой:

```bash
python bot.py
```

Если всё настроено правильно, вы увидите:

```
✅ База данных инициализирована
✅ Бот успешно запущен и готов к работе!
```

### 6. Тестирование

1. Откройте Telegram
2. Найдите вашего бота по username (например: `@Assist_SupportBot`)
3. Нажмите `/start`
4. Проверьте работу всех кнопок

## Статистика

Бот собирает статистику в базу данных SQLite (`database/bot_database.db`):

- Информация о пользователях
- Выбранные тарифы
- Действия пользователей

Для просмотра статистики можно использовать SQLite браузер или написать скрипт.

## Деплой (публикация бота)

### Локальный запуск

Бот работает пока запущен скрипт `bot.py`. При закрытии терминала бот остановится.

### Деплой на сервер

Для работы бота 24/7 нужно разместить его на сервере:

**Варианты:**
- Yandex Cloud (грант 4000₽)
- Timeweb / Reg.ru VPS (~200-500₽/мес)
- Railway.app (бесплатно с ограничениями)

Инструкции по деплою будут предоставлены отдельно.

## Изменение текстов

Все тексты находятся в файле [texts/messages.py](texts/messages.py).

Отредактируйте нужные сообщения и перезапустите бота.

## Поддержка

При возникновении проблем проверьте:
1. Правильность токена в файле `.env`
2. Установлены ли все зависимости (`pip install -r requirements.txt`)
3. Версию Python (`python --version` должен быть 3.8+)
